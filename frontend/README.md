# Frontend (React + Vite)

Планируемый стек: React с Vite и TypeScript. Этот каталог содержит каркас для последующей реализации интерфейсов оператора и администратора (шаги 9+).

## Структура (на текущем этапе)
- `src/` — исходники приложения (будут добавлены на этапе шагов 9–10).
- `package.json` — базовая конфигурация проекта и скриптов.
